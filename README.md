# 🎨 Frontend - Full Stack Project

This is the **frontend application** of a full-stack web project built using **React.js**. It provides a simple and responsive user interface and connects with the backend using REST APIs to perform CRUD operations.

---

## 🧠 About the Project

This frontend handles the user interface of the application. It allows users to interact with data through a clean UI, while communicating with the backend for all data operations.

---

## 🛠️ Tech Stack

- React.js  
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Axios / Fetch API  

---

## 🚀 Features

- Responsive and clean UI  
- API integration with backend  
- CRUD functionality (Create, Read, Update, Delete)  
- Component-based architecture  
- Fast and interactive user experience

## 🔗 Backend Connection

The frontend communicates with the backend using API calls:

axios.get("http://localhost:5000/api/items");

You can also configure environment variables:

REACT_APP_API_URL=http://localhost:5000
